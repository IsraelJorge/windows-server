Esta é a primeira tela do Gerenciador do Servidor, onde realizaremos diversas configurações.

  • Local Server: Aqui, você verá informações importantes sobre o servidor, como o nome, data e hora, status das atualizações, configurações do firewall, entre outros.
  • All Servers: Esta seção exibe uma lista de todos os servidores conectados à rede.
  • File and Storage Services: O compartilhamento de arquivos é uma funcionalidade integrada ao servidor. No entanto, é recomendável evitar usar as configurações padrão e fazer ajustes conforme necessário posteriormente.

2.1 - Alterar o nome do servidor

  1. Inicie o Gerenciador do Servidor (Server Manager).
  2. Vá para o menu "Local Server".
  3. Clique no nome do computador. Isso abrirá uma janela para configurar as propriedades do sistema.
  4. Insira uma descrição para o computador.
  5. Clique em "Change". Isso abrirá outra janela para modificar o nome do computador.
  6. Digite o novo nome do computador.
  7. Clique em "OK" e depois em "Apply" para confirmar as alterações. O sistema solicitará um reinício; clique em "'Restart'" para concluir o processo.

2.2 - Atualizando o sistema

  1. Abra o menu Iniciar clicando no ícone do Windows no canto inferior esquerdo da tela ou pressionando a tecla do logotipo do Windows no teclado. No menu Iniciar, clique em "Settings". Isso abrirá o aplicativo Configurações do Windows.
  2. Dentro das Configurações, role para baixo e clique em "Update and Security".
  3. Na seção "Update and Security", clique na opção "Windows Update" no painel esquerdo.
  4. No painel direito, você verá informações sobre as atualizações disponíveis e o status do seu sistema em relação às atualizações. Clique em "Check updates" para iniciar manualmente a verificação por novas atualizações.
  5. O Windows Server 2016 começará a procurar por atualizações disponíveis. Se houver atualizações pendentes, você verá uma lista delas.
  6. Para baixar e instalar as atualizações, clique em "Baixar" ou "Baixar e instalar" dependendo das opções disponíveis para cada atualização listada.

2.3 - Configuração do IP

Para configurar o IP do servidor como estático, é preciso primeiro desativar o modo DHCP. Siga estes passos para acessar as configurações de IP:

  1. Abra as configurações de rede do servidor.
  2. Procure pela opção que controla o modo DHCP e desative-a.
  3. Depois de desativar o DHCP, você poderá configurar manualmente o IP.
  4. No Gerenciador do Servidor, clique em "Local Server" no painel esquerdo.
  5. Na seção "Properties", ao lado de "IP Address", clique em "IPv4".
  6. Isso abrirá a janela "Local Area Connection Properties" ou "Internet Protocol Version 4 (TCP/IPv4) Properties".
  7. Na janela de propriedades, selecione a opção "Use the following IP address.".
  8. Insira o endereço IP estático desejado no campo "IP Address", a máscara de sub-rede no campo "Subnet mask" e o gateway padrão no campo "Default gateway".
  9. Clique em "OK" para aplicar as alterações e fechar a janela.

Após configurar o IP estático, você poderá acessar o servidor por meio do endereço IP que você definiu.
